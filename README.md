# The Snorlax Application

The Snorlax Application is a simple API app to validate E2E data from [DocKing](https://github.com/shipsaas/docking).

## Tech Stack

- NodeJS 16.x
- SQLite

## Development

```bash
# One time only
cp .env.example .env # also update the ENVs too
touch db.sqlite

# Run server
npm run dev
```

## Production

```bash
npm run prod
```
